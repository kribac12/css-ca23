# FetchFriends

## JavaScript 2 - Course Assignment

FetchFriends is a social media site where users can view, post, edit, and delete posts using a provided API and its documentation.

## 🌟 Features
- View posts
- Create new posts
- Edit existing posts
- Delete posts

## 🛠️ Built With
- HTML
- Bootstrap / SCSS
- JavaScript
